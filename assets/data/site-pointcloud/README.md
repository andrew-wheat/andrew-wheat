# Atlas Site Point Context

Optional generated point-context files can live here as:

- `[slug]-context-points.json`

These files are provisional coordinate-centered context generated from public map data. They are not exact site boundaries, surveys, true LiDAR, or project footprints. If exact future files are added, use those to crop or emphasize the context rather than pretending this provisional radius is a lot line.

The browser should lazy-load these files only after a site is opened in the Atlas tester.
